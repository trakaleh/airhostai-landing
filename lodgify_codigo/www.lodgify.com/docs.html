Getting started

Suggest Edits

The Lodgify public endpoints are offered to our users to facilitate the use of our technology and allow seamless integrations with their applications. In order to ensure a quality experience we are closely monitoring the usage and we expect our users to follow the usage guidelines and respect the rate limits.

All integrations must comply with the Lodgify Terms of Service

Further Information
The Lodgify API allows the user to perform certain operations through REST requests, such as creating new bookings or obtaining a list of them.

All requests and responses only accept and return JSON format.
